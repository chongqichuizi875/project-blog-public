## 1. 背景与目标：将 trust region 归结为 residual 控制

RLVR（带可验证奖励的强化学习）是提升 LLM 推理能力的标准做法：策略生成回复、verifier 给出标量奖励、再以 PPO/GRPO 式的 token 级目标更新模型。训练本质上是 **off-policy** 的——每批回复由固定的 rollout 策略 $\mu$ 采样、复用若干个梯度步，被优化的策略 $\pi$ 因此持续偏离 $\mu$。无约束的更新会破坏训练稳定性、损害推理性能。

抑制此漂移的经典手段是 **trust region**：约束 $\pi$ 相对 $\mu$ 的偏离幅度。现有方法的差异仅在 token 级散度的**度量方式**：

- **PPO / GRPO** 裁剪采样 token 的 likelihood ratio $\rho_t = \pi(y_t\mid s_t)/\mu(y_t\mid s_t)$。单 token 比率是真实散度的 single-sample Monte-Carlo 估计，方差高、且在长尾词表上标定不良。
- **DPPO** 改用 token 级分布散度 $D_t = D_{\mathrm{TV}}(\mu(\cdot\mid s_t),\,\pi(\cdot\mid s_t))$，直接刻画下一 token 分布的变化，而非单次采样结果。

优化的终极目标是 **monotonic improvement** $J(\pi) \ge J(\mu)$。自回归分解给出精确的 finite-horizon 性能差分恒等式（记 $\rho_{a:b} = \prod_{j=a}^{b}\rho_j$）：

$$
J(\pi) - J(\mu) = L'_\mu(\pi) - \Delta(\mu, \pi),
$$

其中 token 级 surrogate 目标 $L'_\mu(\pi) = \mathbb{E}_\mu\big[R\sum_t(\rho_t - 1)\big]$ 为实际被优化的量，而

$$
\Delta(\mu, \pi) = \mathbb{E}_\mu\Big[R\sum_{t=1}^{T}(\rho_t - 1)\,(1 - \rho_{t+1:T})\Big]
$$

是 surrogate 丢弃的 **suffix correction 残差**。恒等式表明：保证单调提升的关键不在于约束散度本身，而在于**极小化残差 $|\Delta|$**。残差中的因子 $1 - \rho_{t+1:T}$ 已指明症结——位置 $t$ 的偏差会进入其后**每一个** token 的条件，此即 §2 两个缺陷的共同来源。

## 2. Residual bound 揭示的两个结构性缺陷

设逐 token 散度满足阈值轮廓 $D_t \le \ell_t$，记 $u_t := \mathbb{E}[D_t]$ 为位置 $t$ 的期望散度。对 suffix likelihood ratio 施加 maximal coupling，残差有如下上界：

$$
|\Delta(\mu, \pi)| \;\le\; 4\xi \sum_{t=1}^{T-1} u_t \sum_{j=t+1}^{T}\ell_j \;\le\; \sum_{t=1}^{T-1}\lambda_t u_t, \qquad \lambda_t := 4\xi\,\bar\ell\,(T - t),
$$

其中 $\xi$ 为奖励绝对值上界、$\bar\ell$ 为单步散度阈值上界。该上界揭示 position-independent 统一阈值（DPPO 强制 $u_t \le \delta$）的两个结构性缺陷。

**缺陷一 · 放大系数的自回归不对称。** 单步偏差 $u_t$ 的放大系数 $\lambda_t \propto (T-t)$ 随 **remaining horizon 线性增长**。早期偏差（$t$ 小）对应极大的系数：单次早期偏差令整条生成 prefix 的分布整体偏移，扭曲其后**所有** token 的条件；晚期偏差的下游影响则可忽略。position-independent 的统一阈值对二者不加区分，因而**低估早期偏差、同时过度限制晚期探索**。

**缺陷二 · 最坏情况下的二次累积。** 逐点独立约束 $u_t \le \delta$ 仅能给出 $\sum_t\lambda_t u_t \le \delta\sum_t\lambda_t$，其允许的最坏情形是每一步偏差均达到上界 $\delta$。代入 $\sum_{t<T}\lambda_t = 2\xi T(T-1)\bar\ell$，残差随长度呈二次膨胀 $O(T^2\delta^2)$。然而相邻步的散度预算本可相互调剂、无需逐步都取最大漂移——逐点独立约束未能利用这一自由度。

## 3. 两个耦合机制：position weighting + prefix-mean budget

两个缺陷同源于「逐 step 孤立、等额约束」。修复之道在于重构约束的**结构**，而非改变散度的度量方式——对应两个缺陷各引入一个机制。

**机制一 · position weight $w_t$（对应缺陷一）。** 引入随位置**单调递减**的权重 $w_t \in [w_{\min}, 1]$，将约束施加于加权散度：

$$
w_t D_t \le \delta.
$$

它等价于一个 **position-dependent** 的 token 级阈值 $D_t \le \delta / w_t$：阈值在序列开头为 $\delta$、至结尾放松至 $\delta / w_{\min}$。据此，$\lambda_t$ 最大的早期位置仅允许更小的 $D_t$，$\lambda_t$ 最小的晚期位置获得放松——散度额度**沿 $\lambda_t$ 由大到小的方向分配**，恰好抵消缺陷一的不对称。

**机制二 · prefix-mean budget $\delta_b$（对应缺陷二）。** 将逐点独立约束升级为对**任意 prefix** 的加权均值约束：

$$
\frac{\sum_{j=1}^{m} w_j D_j}{\sum_{j=1}^{m} w_j} \le \delta_b.
$$

它不再允许每步独立取满，而是强制相邻步**共享**散度预算，直接封闭缺陷二的最坏累积路径。为在线执行该约束，记加权 prefix 散度与累积权重

$$
S_t = \sum_{j=1}^{t} w_j D_j, \qquad W_t = \sum_{j=1}^{t} w_j, \qquad S_0 = W_0 = 0,
$$

则 prefix-mean 约束即写作 $S_m \le \delta_b W_m$。

**两机制的耦合。** $w_t$ 承担双重角色：既缩放单步约束（机制一），又作为 prefix 累积中每步偏差的分配权重（机制二的 $S_t, W_t$）。因此位置越靠前的偏差，在 prefix budget 中占的权重越大——两机制指向同一方向：**收紧早期、约束累积**。

## 4. 参数形式的数学依据：linear $w_t$ 与初始松弛的必要性

机制一、二确定了约束的**结构**，但 $w_t$ 的具体函数、以及在线执行的具体阈值须由收敛要求反推——既非任取一个递减函数，亦非直接照搬 prefix-mean 约束。

### 4.1 $w_t$ 为何取 linear schedule

§6 的收敛证明依赖 Abel summation，其成立要求比例系数 $r_t := \lambda_t / w_t \propto (T-t)/w_t$ **关于 $t$ 单调不增**（否则展开式各项符号无法逐一确定）。这将「$w_t$ 取何形式」转化为一个明确的数学条件。linear schedule

$$
w_t = 1 - \frac{1 - w_{\min}}{T-1}(t-1)
$$

满足该条件——相邻两步之差为

$$
\frac{T-t}{w_t} - \frac{T-(t+1)}{w_{t+1}} = \frac{w_{\min}}{w_t\,w_{t+1}} > 0,
$$

严格为正，即 $r_t$ 严格递减。linear schedule 是满足此收敛条件的最简函数形式。

### 4.2 在线约束为何需要初始松弛 $\delta$

prefix-mean 约束 $S_m \le \delta_b W_m$ 若在线严格执行，会在首个 token（$t=1$）处强制 $D_1 \le \delta_b$。而 $\delta_b$ 极小（默认约 0.02），将扼制生成起步阶段的探索。解法是引入一个**单步尺度的初始松弛** $\delta$，将约束放宽为 $S_t \le \delta + \delta_b W_{t-1}$。据此，在线执行的**有效阈值**取 token 级条件与 prefix 条件中的更严者：

$$
w_t D_t \;\le\; c_t^{\mathrm{CPPO}} := \min\!\Big\{\, \delta,\;\; \delta + \delta_b W_{t-1} - S_{t-1} \,\Big\}.
$$

其行为是 **dynamic** 的：$t=1$ 时 $W_0 = S_0 = 0$，阈值退回 $\delta$，保全起步探索；随生成推进，一旦历史累积漂移 $S_{t-1}$ 超出 prefix budget $\delta_b W_{t-1}$，prefix 项 $\delta + \delta_b W_{t-1} - S_{t-1}$ 跌至 $\delta$ 以下、自动收紧当前 token 的额度。**trust region 随 prefix 漂移累积而动态收紧**，而非对每个 token 等额分配。该松弛在最终残差界中仅引入低阶项 $O(T\bar\ell\delta)$，不影响收紧结论（§6）。

## 5. 合成为单个 token-level mask

两个条件合并为单个逐 token 判据 $I_t$，嵌入标准的 token 级 ratio-advantage 目标（advantage 仍采用 GRPO 的 group-relative advantage）：

$$
M_t^{\mathrm{CPPO}} = \mathbf{1}\!\big[\, \hat A_t(\rho_t - 1) \le 0 \;\;\vee\;\; I_t \,\big], \qquad I_t:\ w_t D_t \le \delta \ \wedge\ S_t \le \delta + \delta_b W_{t-1}.
$$

第一项保留所有将 $\pi$ 拉近 $\mu$ 的更新；$I_t$ 仅约束将 $\pi$ 推离 $\mu$ 的项（判据形式见开篇 [概览图](#fig-overview) 左）。$S_t, W_t$ 均为普通 prefix sum，批量实现仅需沿回复维一次 cumsum + 一次右移一位，**无额外前向、无新 loss 项**。CPPO 由此与 DPPO 在工程上几乎零差异，从而保证后文对比严格隔离出「散度如何分配」这一唯一变量。

## 6. Abel summation：收紧 monotonic improvement bound

### 6.1 证明

prefix-mean budget 等价于：去中心化 prefix slack $\tilde S_m := \sum_{j=1}^{m} w_j (u_j - \delta_b) \le 0$（$m = 1,\dots,T-1$）。将残差界与其常数项作差，用 Abel summation（离散分部求和）改写，令 $r_t = \lambda_t / w_t$：

$$
\sum_{t<T}\lambda_t u_t - \delta_b\sum_{t<T}\lambda_t \;=\; r_{T-1}\tilde S_{T-1} + \sum_{t=1}^{T-2}(r_t - r_{t+1})\,\tilde S_t.
$$

右侧每一项均为「非负系数 $\times$ 非正 slack」：系数 $r_t - r_{t+1} \ge 0$ 由 §4.1 的单调性保证，slack $\tilde S_t \le 0$ 由 prefix budget 保证。故整体 $\le 0$，得

$$
\sum_{t<T}\lambda_t u_t \;\le\; \delta_b\sum_{t<T}\lambda_t \;=\; 2\xi T(T-1)\,\bar\ell\,\delta_b.
$$

Abel summation 将「逐点取满」的最坏路径替换为「prefix-mean 受限」的约束族——$\lambda_t$ 越大的早期位置被压得越紧，残差界中逐点的 $\delta$ 由更小的 prefix-mean $\delta_b$ 取代（$u_t \equiv \delta_b$ 时取等，故对此约束族为紧）。由此得 CPPO 的 monotonic improvement bound：

$$
J(\pi) - J(\mu) \;\ge\; L'_\mu(\pi) - 2\xi T(T-1)\,\bar\ell\,\delta_b.
$$

### 6.2 与同期工作的残差常数对照

把 $\sum_{t<T}\lambda_t = 2\xi T(T-1)\bar\ell$ 代入，各方法的残差常数 $C$ 为：

- **vs DPPO（uniform token-level trust region）**：DPPO 将 $\delta$ 同时用作 $\bar\ell$ 与 $u_t$ 的上界，常数为 $\delta^2$；CPPO 保留同一 $\bar\ell=\delta$，但 prefix 约束将 $u_t$ 项之和压至 $\delta_b$，得 $\delta\cdot\delta_b$。二者之比 $C_{\mathrm{CPPO}}/C_{\mathrm{DPPO}}=\delta_b/\delta$，在 $\delta_b<\delta$ 时严格更紧。
- **vs TRM-Max**：$\max_t D_t\le\delta$ 蕴含每个 $u_t\le\delta$，与 DPPO 落至完全相同的二次常数 $\delta^2$——序列级取 max 无法改善逐点最坏累积。
- **vs TRM-Avg**：仅在**整条回复的终端**约束一次均值，等价于只保留最后一个 prefix 约束。直接计算给 $C_{\mathrm{terminal}}\le 4\xi\bar\ell(T-1)^2\delta_b$，而 CPPO（$w\equiv1$ 时）用上**每一个** prefix 得 $2\xi T(T-1)\bar\ell\delta_b$，二者之比为 $2-\tfrac{2}{T}$——即终端约束最坏松约 2 倍，因其允许早期 prefix 大幅超标、再于后段拉回均值。CPPO 在每个 $m$ 均强制 $S_m\le\delta_b W_m$，杜绝此「早超标、晚补偿」。

> 关键判断：三条对照表明，CPPO 的更紧并非源于更小的阈值或更强的散度度量，而是源于**将约束施加于每个 prefix 的加权均值**——Abel summation 正是把这一约束结构转化为更小残差常数的工具。position weighting 部分进一步给出 $C_{\mathrm{CPPO}}/C_{\mathrm{uniform}}\le \delta_b/(\delta w_{\min})$，其中 $1/w_{\min}$ 是放松晚期阈值的代价。理论收紧的方向（约束累积漂移、对早期收紧）与 §7 实测增益在大模型 / 长 horizon 上最显著、与 DPPO 差距随训练扩大的现象互为印证。

## 7. 实验：跨 Base/post-trained、dense/MoE 的一致提升

- **全设置一致最佳，且大模型增益最大**：CPPO 相比 GRPO、CISPO、MinPRO、DPPO、TRM 等 baseline，在跨 Base/post-trained、dense/MoE 的各设置上均取得最佳，较次优的绝对领先分别为 **+3.06 / +0.91 / +1.39 / +5.56**。最大增益出现在 **Qwen3-30B-A3B-Base**——唯一的 16k rollout 设置，亦即 §2「remaining-horizon 放大」最显著的区域（早期 token 偏差须穿越更长的 suffix 传播）。
- **与 DPPO 的对照严格受控**：二者共用同一套 Top-K（K=20）reduced-TV 散度与同一 per-model 阈值尺度 $\delta$，**仅在 weighting 与 prefix 约束上不同**。此匹配设置下 CPPO 较 DPPO **+3.69 / +1.88 / +2.22 / +5.56**——增益确定源于「散度沿回复的分配方式」，而非更优的散度度量或更松的阈值。

## 8. 消融：增益源于「分配」而非「度量」

在 Qwen3-1.7B 与 1.7B-Base 上，从匹配的 DPPO 基线出发逐步加回 CPPO 组件，隔离每个机制的贡献。

三条结论收敛于同一点：CPPO 的增益**源于散度预算沿回复的分配方式（position order + prefix 累积），而非散度的度量方式或近似粒度**。这与 §6 的理论一致——收紧的是残差常数中「逐点 $\delta$ → prefix-mean $\delta_b$」这一项，与采用 TV 抑或 KL、Top-K 抑或 Binary 无关。
